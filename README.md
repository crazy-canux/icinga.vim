# icinga.vim

Icinga syntax for vim. 

Inspired by <https://github.com/Icinga/icinga2/tree/master/tools/syntax/vim>

# How to install

## pathogen

    cd ~/.vim/bundle
    git clone git@github.com:crazy-canux/icinga2.vim.git
    
## vundle

    vi ~/.vimrc
    Plugin 'crazy-canux/icinga2.vim'
