# icinga.vim

[Icinga](https://www.icinga.org) syntax for [vim](http://www.vim.org/).

Inspired by <https://github.com/Icinga/icinga2>

# How to install

## pathogen

    cd ~/.vim/bundle
    git clone git@github.com:crazy-canux/icinga2.vim.git

## vundle

    vi ~/.vimrc
    Plugin 'crazy-canux/icinga2.vim'
